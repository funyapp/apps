# PassFM

**PassFM** is a file manager application, you can setup password to protect your privacy: photos, videos, documents, notes, etc.

## Tips⚠️

1. **All data is stored only on the local device and is not uploaded to the server.**
2. **Please backup your data regularly, to prevent accidental data loss.**
3. **Please do not remove/uninstall this app, or you will lose all your data.**

## Setup/Reset access password

`App` - `Homepage` - `Menu` on nav-bar - `Settings` -> `Reset Password`.

## File Manager

### Import/Export

#### Import from Photos

1. App - `Albums` - `More` button on nav-bar - `New` - Input album name - `OK`.
2. Go into the album you just created - `More` button on nav-bar - `Import` - Select photos - `Done`.
3. Import videos is similar.

#### Import via USB

1. Connect iPhone to computer with USB cable.
2. Open Finder - My Device - File Sharing - PassFM.
3. Select files from computer and add to File Sharing.

#### Export to Photos/Files

1. In the specified album or video folder - `Select` - `Share` - `Save Images` or `Save to Files`.

#### Import/Export via WiFi

1. Please ensure that your iPhone and computer have connected to the same Wi-Fi
2. Please allow the app and your web browser to **access your local network**.
3. App - `Albums` - `More` button on nav-bar - `WiFi` - `Start` button.
4. Open browser on computer and visit the app given address.
5. Drag files to the browser, or download files from browser.

### Delete/Rename

- Delete Album: `Albums List` - `Swipe to Left` - `Delete`.
- Rename Album: `Albums List` - `Long Press` - `Rename`.
- Delete Photo: `Album Detail` - `Edit` - `Select` - `Delete`.
- Delete Video Folder: `Videos Folder List` - `Edit` - `Select` - `Delete`.
- Rename Video Folder: `Videos Folder List` - `Long Press` - `Rename`.
- Delete Video: `Video Folder Detail` - `Swipe to Left` - `Delete`.
- Notes & Docs: `Swipe to Left` to Delete, `Long Press` to Rename.

## Web Browser

With the built-in browser, you can browse the web securely without worrying about your browsing history being leaked. You can also easily download images and videos from web pages, or save web pages as PDF files.
